# PlayingCard_War

トランプゲーム「戦争（War）」をプレイできるJava製GUIアプリケーションです。
Java Swing を使用してグラフィカルな画面でカードゲームを楽しむことができます。



https://github.com/user-attachments/assets/205f714a-0eae-4439-91dc-06355bf37c60



## 特徴
- **2人対戦モード対応**: 現在、2人での対戦モードが完成しており、以下の組み合わせで遊ぶことができます。
  - プレイヤー vs プレイヤー
  - プレイヤー vs CPU
  - CPU vs CPU
- **直感的な操作**: 画面上の山札をクリックすることでカードを引き、勝敗を判定します。

※ **3人以上の対戦モードについて**
現在、3人以上のプレイ人数を選択した場合は「準備中」の画面が表示されるようになっています。

## 実行環境
- Java Runtime Environment (JRE) 8 以上
- OS: Windows / macOS / Linux (Javaが動作する環境)

## 実行方法
1. 本プロジェクトをEclipseなどのIDEにインポートします。
2. `src/Main.java` コアプログラム実行クラスを右クリックし、「Run As」>「Java Application」を選択して実行します。
3. または、コマンドラインからコンパイルして実行することも可能です。
   ```bash
   cd src
   javac Main.java
   java Main
   ```

## 遊び方
1. ゲームを起動すると、プレイ人数とCPUの数を設定する画面が表示されます。
   - 合計が2人になるように設定してください。（例: Player 1人、CPU 1人）
2. ゲームが開始されたら、自分のターン（あるいは両者同時）に山札をクリックしてカードを出します。
3. ダイアログ等で勝敗が判定され、勝った方が場のカードを獲得します。
4. 終了時の獲得枚数で勝敗が決定します。画面の指示に従ってプレイしてください。
